\section{Prompts Clasificacion}\label{sec:prompts}
Estructura de los 24 prompts utilizados en la clasificacion de textos. La notacion y el conocimiento linguistico utilizado es la siguiente:

\begin{itemize}
    \item \proriginaltextC Ejemplos de textos originales de cada nivel, seleccionados y extraídos de los datasets de Kaggel.
    \item \prdescriptorsN{A1}, \prdescriptorsN{A2}, etc. Descriptores del nivel CEFR objetivo, según lo establecido por {CouncilEurope2020} [cita]
    \item \prlistN{A1}, \prlistN{A2}, etc. Lista de vocabulario correspondiente a cada nivel, provista por {Oxford3000} [cita].
\end{itemize}

\subsection{Base: 0-shot prompts}

\begin{prompt}
    [You are an English teacher and I want you to classify the following text according to the CEFR classes. I want you to respond only with the class (A1|A2|B1|B2|C1|C2). Without introduction.]
{Prompt 0 (sin lectocomprecion)}
\end{prompt}

\begin{prompt}
    [You are an English teacher and I want you to classify the following text for reading comprehension according to the CEFR classes. I want you to respond only with the class (A1|A2|B1|B2|C1|C2). Without introduction.]
{Prompt 1 (con lectocomprecion)}
\end{prompt}

\begin{prompt}

    I detect that you have a bias to classify everything as B2, correct it.]
{Prompt 2 (Prompt 0 + corrección de sesgo B2)}
\end{prompt}

    [this is a list of CEFR level descriptors for level A1:\\
    \prdescriptorsN{A1}\\
    this is a list of CEFR level descriptors for level A2:\\
    \prdescriptorsN{A2}\\
    this is a list of CEFR level descriptors for level B1:\\
    \prdescriptorsN{B1}\\
    this is a list of CEFR level descriptors for level B2:\\
    \prdescriptorsN{B2}\\
    this is a list of CEFR level descriptors for level C1:\\
    \prdescriptorsN{C1}\\
    this is a list of CEFR level descriptors for level C2:\\
    \prdescriptorsN{C2}\\
    Using the above descriptors I want you to classify the following text according to its Common European Framework of Reference (CEFR) level. I want you to respond only with the class (A1|A2|B1|B2|C1|C2). Without introduction.]
{Prompt 21 (descriptores de cada nivel)}
\end{prompt}
